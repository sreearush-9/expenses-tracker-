Expense Tracker - Project Report
Generated: 2026-03-25

1. Project Overview
- Stack: Node.js, Express 4, Sequelize ORM, PostgreSQL, JWT, bcryptjs, React 19, Vite 8
- Purpose: Secure expense tracking app with login/register and user session via
  JWT.

2. Core Features Implemented
- User registration and login (JWT authentication).
- Protected expense APIs (CRUD style): add, view, edit, delete.
- Dashboard with:
    - Total expenses
    - Expense list
    - Pie chart distribution
    - Top 5 expenses
    - Add/Edit/Delete actions
    - Filters (search, amount range, sort)
    - Export CSV
- Separate UI pages:
    - /  (landing page)
    - /login
    - /register
    - /dashboard (redirect to login if not authenticated)

3. Authentication Flow
- POST /api/auth/register -> creates user and returns JWT.
- POST /api/auth/login -> validates user and returns JWT.
- Frontend stores token in localStorage and sends Authorization: Bearer <token>.
- If token is missing/expired, dashboard redirects to login.

7. API Summary

8. Run Instructions
1) Install dependencies:
   cd backend && npm install
   cd frontend && npm install
2) Start backend:
   cd backend && node server.js
3) Start frontend:
   cd frontend && npm run dev
4) Open:
   http://localhost:5173/
5) Register or Login

- CORS is configured to allow only the frontend origin.
